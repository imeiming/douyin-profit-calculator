# 抖音利润计算器使用示例

## 示例1：纯文本表格输入

**用户输入：**
```
商品名称 | 销售数量 | 单价 | 退款数量
连衣裙 | 150 | 89 | 8
T恤 | 300 | 45 | 15
牛仔裤 | 200 | 120 | 12
```

**预期输出：**
- 总销售额、总成本、净利润、利润率
- 详细的成本构成分析
- 各商品利润明细
- 优化建议

## 示例2：JSON格式输入

**用户输入：**
```json
[
  {"product": "连衣裙", "quantity": 150, "price": 89, "refunds": 8},
  {"product": "T恤", "quantity": 300, "price": 45, "refunds": 15},
  {"product": "牛仔裤", "quantity": 200, "price": 120, "refunds": 12}
]
```

## 示例3：简单CSV输入

**用户输入：**
```
连衣裙,150,89,8
T恤,300,45,15  
牛仔裤,200,120,12
```

## 自定义成本结构

**用户输入：**
```
商品名称 | 销售数量 | 单价 | 退款数量
连衣裙 | 150 | 89 | 8

商品成本按35%计算，物流成本每单10元
```

**处理逻辑：**
- 解析销售数据
- 提取自定义成本参数
- 应用新的成本结构重新计算
