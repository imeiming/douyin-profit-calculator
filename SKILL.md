# 抖音利润计算器

## 概述

抖音利润计算器是一个专门用于计算抖音电商销售利润的技能。当用户发送销售报表数据时，此技能会自动触发并执行以下功能：

1. **销售报表解析** - 解析用户提供的销售数据
2. **利润计算逻辑** - 应用预设的成本结构和费率计算净利润  
3. **成本分析** - 提供详细的成本构成分析

## 触发条件

当用户消息包含以下关键词时自动触发：
- "销售报表"
- "利润计算" 
- "抖音销售"
- "电商数据"
- 包含数字表格或销售数据

## 输入格式

支持多种输入格式：

### 1. 纯文本表格
```
商品名称 | 销售数量 | 单价 | 退款数量
商品A | 100 | 50 | 5
商品B | 200 | 30 | 10
```

### 2. JSON格式
```json
[
  {"product": "商品A", "quantity": 100, "price": 50, "refunds": 5},
  {"product": "商品B", "quantity": 200, "price": 30, "refunds": 10}
]
```

### 3. CSV格式（单行或多行）
```
商品A,100,50,5
商品B,200,30,10
```

## 成本结构

默认成本结构包括：

| 成本项 | 费率/说明 |
|--------|-----------|
| 商品成本 | 单价的40% |
| 抖音佣金 | 销售额的5% |
| 支付手续费 | 销售额的1% |
| 物流成本 | 每单8元 |
| 退货损失 | 退款金额的100% |
| 营销费用 | 销售额的3% |

## 输出格式

返回详细的利润分析报告，包含：

1. **总销售额**
2. **总成本明细**  
3. **净利润**
4. **利润率**
5. **各商品利润分析**
6. **优化建议**

## 使用方法

直接发送销售报表数据，技能会自动处理：

```
商品名称 | 销售数量 | 单价 | 退款数量
连衣裙 | 150 | 89 | 8
T恤 | 300 | 45 | 15
```

## 自定义成本

用户可以通过以下方式自定义成本结构：

- "商品成本按35%计算"
- "物流成本每单10元"  
- "抖音佣金按8%计算"

## 技术实现

使用Python脚本 `scripts/calculate_profit.py` 处理数据，不直接读取外部表格文件，而是解析用户提供的文本数据。

## 错误处理

- 数据格式错误：提示正确的输入格式
- 缺少必要字段：要求补充完整信息
- 计算异常：返回错误详情和建议
